## 📚 Phạm Vi Thi – Toàn Bộ CSB (Buổi 1 → 11)

### Nhóm 1: Python Cơ Bản (Buổi 1–2)
- Biến, kiểu dữ liệu, toán tử
- `if/elif/else`, `for`, `while`
- List: tạo, truy cập, thêm/xóa, sort, slice
- Hàm: `def`, tham số, `return`, giá trị mặc định
- File: `open()`, `with`, `r`/`w`/`a`

### Nhóm 2: OOP (Buổi 3–4)
- Class, Object, `__init__`, `self`
- Private (`__`), getter, setter
- Kế thừa, `super()`
- Đa hình (override), `isinstance()`
- Abstraction (`ABC`, `@abstractmethod`)

### Nhóm 3: Thuật Toán (Buổi 6–7)
- Linear Search – O(n)
- Binary Search – O(log n), điều kiện dùng
- Bubble Sort – O(n²), cơ chế hoán đổi
- Insertion Sort – O(n²), cơ chế chèn
- Quick Sort – O(n log n), pivot + đệ quy

### Nhóm 4: Cấu Trúc Dữ Liệu Nâng Cao (Buổi 10–11)
- Set: `|`, `&`, `-`, `^`, `in`, `add`, `discard`
- Dictionary: key-value, `.get()`, `.items()`, comprehension
- Stack: LIFO, `list.append/pop`
- Queue: FIFO, `deque.append/popleft`
