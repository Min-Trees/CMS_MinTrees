## 🎯 Mục Tiêu

- Chọn cột và hàng bằng `[]`, `loc`, `iloc`
- Lọc dữ liệu theo điều kiện một hoặc nhiều cột
- Phát hiện và xử lý missing values (`NaN`)
- Thêm/xóa cột, đổi tên cột

## 📚 Nội Dung Chính

1. **Truy xuất dữ liệu** — `df[col]`, `df[cols]`, `df.loc[]`, `df.iloc[]`
2. **Filtering** — boolean mask, `&`, `|`, `~`, `isin()`, `between()`
3. **Missing Values** — `isna()`, `fillna()`, `dropna()`
4. **Thao tác cột** — `rename()`, `drop()`, `assign()`, computed column
