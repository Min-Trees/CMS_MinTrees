📦 1.1 Biến và Kiểu Dữ Liệu

Biến trong Python là nơi lưu trữ dữ liệu, giống như một chiếc hộp có dán nhãn. Bạn đặt tên cho hộp để dễ nhớ và bỏ giá trị vào bên trong.

Quy tắc đặt tên biến

  • ✅ Dùng snake_case: tất cả chữ thường, nối bằng dấu gạch dưới
  • ✅ Tên phải có ý nghĩa: diem_toan, ten_hoc_sinh
  • ❌ Không bắt đầu bằng số: 1diem
  • ❌ Không dấu cách: ten hoc sinh
  • ❌ Không dùng tiếng Việt có dấu

4 kiểu dữ liệu cơ bản

KiểuMô tảVí dụ
intSố nguyêntuoi = 16
floatSố thựcdiem = 8.5
strChuỗi văn bảnten = "An"
boolĐúng/Saila_hs = True

🔀 1.3 Cấu Trúc Điều Kiện

Cấu trúc điều kiện giúp chương trình đưa ra quyết định dựa trên điều kiện. Quy tắc quan trọng: dấu : cuối dòng if là BẮT BUỘC, thụt lề 4 dấu cách cho code bên trong.

diem_tb = 7.5

if diem_tb >= 9.0:
    xep_loai = "Xuất Sắc"
elif diem_tb >= 8.0:
    xep_loai = "Giỏi"
elif diem_tb >= 6.5:
    xep_loai = "Khá"
elif diem_tb >= 5.0:
    xep_loai = "Trung Bình"
else:
    xep_loai = "Yếu"

print(f"Điểm {diem_tb} → Xếp loại: {xep_loai}")

🔄 1.4 Vòng Lặp

Vòng lặp giúp thực hiện công việc lặp đi lặp lại nhiều lần. for duyệt qua từng phần tử, while lặp đến khi điều kiện sai.

⚠️ Cảnh báo: Vòng while nếu quên thay đổi biến điều kiện sẽ dẫn đến vòng lặp vô tận và máy bị treo! 
# Vòng for - in bảng cửu chương
print("Bảng cửu chương 7:")
for i in range(1, 11):
    print(f"  7 x {i:2d} = {7*i}")

# Vòng while - đếm ngược
dem = 5
while dem > 0:
    print(f"  {dem}...")
    dem -= 1
print("  Bắt đầu!")

  • if/elif/else: kiểm tra từ cao xuống thấp, thụt lề 4 dấu cách
  • for duyệt dãy số, while cần thay đổi biến điều kiện

📋 2.1 List - Danh Sách

List là dãy các phần tử có thứ tự, giống dãy ngăn tủ có đánh số từ 0. Index bắt đầu từ 0, có thể chứa nhiều kiểu dữ liệu, có thể thay đổi được.

Cú phápÝ nghĩa
ds[i]Truy cập phần tử thứ i

ds.append(x)Thêm x vào cuối
ds.insert(i, x)Chèn x vào vị trí i
ds.remove(x)Xóa phần tử có giá trị x
ds.pop(i)Xóa và trả về phần tử thứ i

🔧 2.2 Hàm (Function)

Hàm giúp đóng gói code để tái sử dụng. Giống như máy pha cà phê: bỏ nguyên liệu vào (tham số) → xử lý → ra sản phẩm (return).

def tinh_trung_binh(danh_sach):
    """Tính điểm trung bình, trả về 0 nếu list rỗng."""
    if not danh_sach:
        return 0
    return sum(danh_sach) / len(danh_sach)

def xep_loai(diem, he_thang=10):  # Tham số mặc định
    """Xếp loại theo điểm."""
    ty_le = diem / he_thang * 10
    if ty_le >= 9.0:   return "Xuất Sắc"
    elif ty_le >= 8.0: return "Giỏi"
    elif ty_le >= 6.5: return "Khá"
    elif ty_le >= 5.0: return "Trung Bình"
    else:              return "Yếu"
💡 Lưu ý: Tham số có giá trị mặc định phải đứng SAU trong danh sách tham số. Hàm không có return sẽ trả về None.

📂 2.3 Đọc/Ghi File

File giúp lưu dữ liệu vĩnh viễn, không mất khi tắt máy. Luôn dùng with open() và encoding="utf-8" để đọc tiếng Việt đúng.

ModeÝ nghĩa
"r"Đọc (file phải tồn tại)
"w"Ghi mới (XÓA nội dung cũ)
"a"Ghi tiếp vào cuối (giữ nội dung cũ)
 
with open("diem_lop.txt", "w", encoding="utf-8") as f:
    f.write("=== BẢNG ĐIỂM LỚP 10A1 ===\\n")
    for i in range(len(hoc_sinh)):
        f.write(f"{hoc_sinh[i]:<10}: {diem[i]:.1f}\\n")

with open("diem_lop.txt", "r", encoding="utf-8") as f:
    print(f.read())

📝 Tổng kết chương 2

  • List index bắt đầu từ 0, dùng append/insert/remove/pop để thay đổi
  • Hàm = máy tái sử dụng: def + return, tham số mặc định đứng sau
  • List comprehension: [x for x in ds if dk] - cú pháp ngắn gọn
  • File: luôn dùng with open() + encoding='utf-8' cho tiếng Việt
