# MindX Course - Python & AI Learning Platform

Học Python từ cơ bản đến chuyên sâu qua 3 khóa học: CSB, CSA, CSI.

## 🎓 Các Khóa Học

### 🐍 CSB - Python Cơ Bản
- **14 bài học** | 90 phút/bài
- Biến, kiểu dữ liệu, toán tử
- Cấu trúc điều khiển (if/elif/else, for, while)
- List, Tuple, Dictionary
- Hàm và xử lý file
- OOP: Class, Kế thừa, Đa hình, Abstract Class
- Thuật toán: Tìm kiếm, Sắp xếp, Stack, Queue

### 📊 CSA - Data Science
- **8 bài học** | 90 phút/bài
- pandas: DataFrame, lọc, groupby
- Trực quan hóa với matplotlib/seaborn
- Machine Learning cơ bản
- Linear Regression, Classification
- Project Data Analysis

### 🤖 CSI - AI & Deep Learning
- **14 bài học** | 90 phút/bài
- Neural Networks: Perceptron, MLP
- TensorFlow/Keras
- CNN cho Computer Vision
- Transfer Learning
- NLP: Tokenizer, Transformers
- Hugging Face
- Object Detection, Speech AI
- Deployment với Streamlit

## 🚀 Cách Sử Dụng

### Cách 1: Mở trực tiếp
Mở file `index.html` trong trình duyệt.

### Cách 2: Local Server
```bash
cd docs
python -m http.server 8000
# Mở http://localhost:8000
```

### Cách 3: Deploy lên GitHub Pages

1. Fork repository này
2. Vào Settings → Pages
3. Chọn branch `main` và folder `/docs`
4. Save và đợi deployment

### Cách 4: Deploy lên Vercel

1. Cài Vercel CLI: `npm i -g vercel`
2. Chạy: `vercel --prod`
3. Hoặc kéo thả folder `docs` lên [vercel.com/new](https://vercel.com/new)
